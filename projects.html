A Fashion Recommendation System

CSCI-351 Spring 2024
Colleagues: Chris Haleas, Sanmeet Singh, and Rehaan Rafi
  • Developed a fashion recommendation system using machine learning models to suggest personalized clothing based on user entered attributes
  • Trained K-Nearest-Neighbors, Random Forest, and Decision Tree models on a Kaggle dataset containing 45,000 clothing items with attributes.
  • Users provide preferences (gender, clothing type, color, season, usage context) to receive personalized recommendations.
  • Cosine similarity between user preferences and predicted items is used to measure recommendation quality.
